Classification of Cotton Leaf Diseases and Pests Using CNN

This project builds a Convolutional Neural Network (CNN) from scratch to classify cotton leaf images into six categories: Bacterial Blight, Powdery Mildew, Target Spot, Aphids, Army Worm, and Healthy Leaves.

Abstract

Cotton is essential for textile and food industries but is vulnerable to diseases and pests. Manual diagnosis is slow and error-prone. This research uses image processing and a custom CNN to classify cotton leaf conditions. The dataset was split into training (75%), validation (15%), and testing (10%). After augmentation and 23 epochs of training (batch size 16), the model achieved 91% accuracy (330/361).

Methodology

  • Data collection & preprocessing: Image resizing, normalization, augmentation.
  • Model: Custom CNN with convolutional, pooling, dropout, and dense layers.
  • Training: Categorical crossentropy, Adam optimizer, early stopping & model checkpoint.
  • Evaluation: Confusion matrix, per-class accuracy, test set evaluation.

Results

Per-class correct predictions: Aphids 58/65, Army Worm 56/60, Healthy Leaves 60/60 (100%), Bacterial Blight 53/60, Target Spot 52/61. Overall 330/361 → 91% final accuracy.

Future Work

  • Expand dataset with more disease/pest classes and diverse field images.
  • Experiment with pre-trained backbones (ResNet, EfficientNet) and object detection (YOLO) for localized detection.
  • Deploy as a mobile or web app for real-time farmer assistance.
